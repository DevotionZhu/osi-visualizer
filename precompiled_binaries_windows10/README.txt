Running OSI Visualizer on Windows 10 OS:

- Author: Monish Gogri ( monish9gogri@gmail.com )
- Date: 19.11.2018
- OS: Windows 10 OS

Running OSI Visualizer:
- Unzip the contents of this directory to a desired location.
- To run the OSI Visualizer click the batch file: "Run_osi-visualizer.bat".
- You may also run the example file to verify the working, provided in this directory called "example_sensor_view.txt". (available on https://github.com/OpenSimulationInterface/osi-visualizer/releases/download/1.0/example_sensor_view.txt)
- Please Note: You will require a minimum OpenGL version of 3.4. If the OpenGL components are not showing on your computer, try installing the latest drivers of your Intel HD Graphics or your graphic card.

Updated stand alone package: https://syncandshare.lrz.de/dl/fiFJSw9aXWgbDhhWwQKvGJKS/OSI_Visualizer.zip
